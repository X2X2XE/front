함수(Function) : 특정 기능을 수행하기 위한 코드 실행기 - 호출되어야 만 작동

  • 일반적으로 Function 이라는 키워드를 사용하여 선언
  • function 키워드 뒤에 기능을 함축설명하는 의미의 함수명을 작성
  • 함수명 뒤에 ()를 작성한 후 선택적으로 파라미터로 사용할 변수를 선언 할 수 있다.
  • 이 때 파라미터의 타입과 형식은 자유도가 높아 다양한 형태로 사용이 가능하다
  • ()뒤에 {}로 함수의 영역을 지정한 후 그 안에 기능 로직을 작성하면 된다.
  • 기능로직에는 return이라는 키워드를 사용하여 함수의 종료 시점을 지정할 수 있다.
  • return은 함수 실행 종료 역할을 함과 동시에 특정 결과 값을 호출한 시점의 코드로 반환 할 수 잇다.
  • 함수의 선언 방법은 크게 3가지를 사용할 수 있는데 사용 목적이 조금씩 차이가 잇기에 숙지해야함
